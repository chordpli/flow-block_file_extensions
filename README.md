# 프로젝트 개요
마드라스 체크(플로우) 과제 전형

## 과제 주제
파일 확장자 차단

## 기술

- 자바 : JAVA 11
- 개발 툴 : SpringBoot 2.7.8
- 필수 라이브러리 : SpringBoot Web, MySQL, Spring Data JPA, Lombok, Spring Security, Spring Cloud Starter, Thymeleaf
- 빌드 : Gradle 7.6
- DB : AWS RDS
- Devops : AWS EC2, AWS S3, Docker
